The codes are organized into two main folders:

- `v16.68 auto order`  
  Simulation codes for the **look direction error** scenario.

- `v16.68_coherent`  
  Simulation codes for the **coherent scattering mismatch** scenario.

These codes are associated with ongoing research on robust adaptive beamforming. The related manuscripts are currently **under peer review**.
